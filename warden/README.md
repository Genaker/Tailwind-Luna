# Warden files (tracked)

This folder holds **versioned** copies of Warden configuration used with **Tailwind Luna** on this Magento project. Runtime installs them at the **Magento project root** (not inside the theme package).

| File | Install location |
|------|-------------------|
| **`warden-env.yml`** | `<magento-root>/.warden/warden-env.yml` |
| **`env.warden.example`** | `<magento-root>/.env` (copy or merge; see `docs/WARDEN.md`) |

Full setup and daily commands: **`docs/WARDEN.md`**. Optional **[WardenGUI](https://github.com/Genaker/WardenGUI)** (`pip install wardengui`) — see **“WardenGUI”** in that doc.
